# CurseForge Description

## Short Description
Automatically purchase class-specific reagents from vendors with intelligent selection and smart buying.

## Full Description

**StockUp** is a World of Warcraft Classic/TBC addon that automatically purchases class-specific reagents when you visit vendors.

### Key Features
- **Automatic Purchases**: Buys reagents automatically at vendor windows
- **Smart Selection**: Automatically picks the best reagent for your spell ranks
- **Smart Buy Mode**: Only purchases what you need to reach your target amount
- **Class-Specific**: Detects your class and only buys relevant reagents
- **Manual Override**: Choose specific reagents if you prefer
- **Custom Amounts**: Set different quantities for each reagent type

### Supported Classes
- Priest (Holy Candle, Sacred Candle)
- Paladin (Symbol of Divinity, Symbol of Kings)
- Shaman (Ankh, Fish Scales, Fish Oil)
- Druid (Seeds, Berries)
- Rogue (Flash Powder)

### Commands
- `/su` - Open configuration
- `/su toggle` - Enable/disable addon
- `/su auto` - Toggle auto-buy
- `/su vendors` - Show vendor locations
- `/su info` - Display character info

### Perfect For
- Casting reagent-heavy spells frequently
- Managing multiple alts with different reagent needs
- Never running out of reagents during raids or dungeons
- Hassle-free reagent management

**Installation**: Extract to AddOns folder and reload UI. That's it!
